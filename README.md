# godot-actionadv-learn

## Overview

This project is a learning initiative focused on building an Adventure Game using Godot Engine. The goal is to explore game development concepts, engine features, and C# scripting within the Godot environment.

## Tech Stack

- **Language**: C#
- **.NET Version**: .NET 10
- **Game Engine**: Godot v4.5.1

## Requirements

To run and develop this project, you'll need:

- Godot Engine v4.5.1 (with .NET support)
- .NET SDK 10 or later
- A code editor (VS Code recommended with C# and Godot extensions)

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/abcamitan/godot-actionadv-learn.git
   cd godot-actionadv-learn
   ```

2. **Open the project in Godot**:
   - Launch Godot Engine v4.5.1
   - Click "Import" and navigate to the project directory
   - Select the `project.godot` file (once created)
   - Click "Import & Edit"

3. **Build the C# solution**:
   - In Godot, go to Project → Tools → C# → Create C# solution
   - This will generate the necessary .csproj files
   - Build the project using the Build button in Godot

4. **Run the game**:
   - Press F5 or click the Play button in Godot to run the game
